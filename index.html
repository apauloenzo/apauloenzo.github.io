Objetivos Gerais da Disciplina

A disciplina de desenvolvimento web tem como objetivo fornecer aos estudantes conhecimentos e habilidades necessárias para projetar, desenvolver e implementar aplicações web de alta qualidade.
Entre os objetivos estão: Compreender os princípios básicos de desenvolvimento web: A disciplina deve fornecer aos estudantes uma compreensão completa dos conceitos fundamentais de desenvolvimento web, como HTML, CSS, JavaScript, APIs e protocolos web.
Desenvolver habilidades de programação: Além de entender a teoria, os estudantes devem ter a oportunidade de colocar em prática seus conhecimentos, desenvolvendo habilidades de programação em diferentes linguagens de programação para web.
Aprender boas práticas de desenvolvimento: A disciplina deve ensinar aos estudantes as melhores práticas de desenvolvimento, incluindo design responsivo, acessibilidade, otimização para mecanismos de busca (SEO) e segurança.
Criar aplicações web completas: Os estudantes devem ter a oportunidade de aplicar seus conhecimentos e habilidades para desenvolver aplicações web completas, desde a fase de planejamento até a implementação final.
Trabalhar em equipe: A disciplina também deve enfatizar a importância do trabalho em equipe e da colaboração para o sucesso de projetos de desenvolvimento web.

Tópicos de Aprendizagem

HTML e CSS: Estudo dos fundamentos do HTML e do CSS, incluindo marcação semântica, layout responsivo, estilos e seletores.
JavaScript: Introdução ao JavaScript, incluindo sintaxe básica, manipulação do DOM, eventos, funções, objetos, AJAX e JSON.
Frameworks de desenvolvimento web: Estudo de frameworks populares de desenvolvimento web, como React, Angular, Vue.js e Node.js.
